Javascript - input(text)物件

名稱 類形 說明
accessKey 屬性 String value that sets/ returns the accessKey for the field.
defaultValue 屬性 元性初始設定的value值
disabled 屬性 Boolean value that sets/ returns whether the field is disabled.
form 屬性 指向表單元素所在的<form>
References the form that contains the text field.
maxLength 屬性
name 屬性 讀取/設定表單元素name屬性的值
Sets or returns the value of the name attribute of a form element.
readOnly 屬性
size 屬性
type 屬性 設定/讀取表單元素type屬性的值；IE8(含)以下的瀏覽器只能讀取，不支援設定屬性的功能。
A property available on all form elements, "type" returns the type of the calling form element, in this case, "text".
value 屬性 存/取表單元素value屬性的值
Read/write string that specifies the value of the text field.
onBlur 事件屬性 Code is executed when the focus is removed from the text field.
onChange 事件屬性 Code is executed when the user changes the value within the text field, and removes focus away from the field.
onFocus 事件屬性 Code is executed when the focus is set on the text field.
onKeyDown 事件屬性 Code is executed when user presses down the key within the text field.
onKeyPress 事件屬性 Code is executed when user presses the key within the text field.
onKeyUp 事件屬性 Code is executed when user releases a key within the text field.
onSelect 事件屬性 Code is executed when user selects some text within the text field.
blur() 方法 使焦點離開某個元素
Removes focus away from the text field.
focus() 方法 聚焦到某個元素上
Sets focus on the text field.
範例： 
<form>
	<input type="text" name="text01">
</form>
<script>
	window.onload = function(){
		document.forms[0].text01.focus();
	}
</script>
select() 方法 選取元素內的文字
Highlights the content of the text field.
範例： 
<form>
	<input type="text" name="input01" value="select" />
</form>
<script>
	document.forms[0].input01.onmouseup = function(){
		this.select();
	}
</script>
