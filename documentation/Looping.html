名稱 類形 說明
for 關鍵字 for迴圈是前測試迴圈(代表是否終止迴圈的條件判斷是在執行內部程式碼之前，因此迴圈的主體可能根本不被執行)，在進入迴圈之前能夠初始化變數，並且定義迴圈後要執行的程式碼，其語法如下：
for(initial var; expression; post-loop-expression){statement}
執行過程如下：
第一個項目(initial var)是用來設定控制迴圈執行次數的變數初值，這個項目只會被執行一次。
第二個項目(expression)是是否執行迴圈的條件判斷式，只有成立，它就會執行大刮號{ ... }裡的程式，直到條件判斷為否為止。
第三個項目(post-loop-expression)是每次執行完{ ... }裡的程式就會執行的敘述，通常是增加或減少控制迴圈執行次數的變數。
for迴圈最常用的型式是for(i=0;i<n;i++){statement;}，它表示迴圈一共執行n次，非常適用於己知迴圈次數的運算。
以下範例將使用for迴圈計算1+2+3+4+5的和。
var sum=0;
for(var i=1; i<=5; i++){
	sum+=i;
}
alert(sum);
巢狀迴圈；如果我們有一個二維陣列，可以在for迴圈中再加入for迴圈列舉二維陣列中所有的值： 
var arr = [[1,2],[3,4],[5,6]];
for (let i = 0; i < arr.length; i++) {
	for (let j = 0; j < arr[i].length; j++) {
		  console.log(arr[i][j]);
	}
}
//輸出結果：1 2 3 4 5 6
while 關鍵字 while迴圈是前測試迴圈，這代表是否終止迴圈的條件判斷是在執行內部程式碼之前，因此迴圈的主體可能根本不被執行，其語法如下：
while(expression){statement;}
當expression為true時，程式會不斷的被執行，直到expression變為false。
如下例會利用i來重覆計算sum的加總：
var i=1;
var sum=0;
while(i<=100){
	sum+=i;
	i++;
}
alert(sum);
do/while 關鍵字 do ... while 迴圈它是一種先斬後奏型的迴圈結構，無論如何 do {...} 之間的程式至少會被執行一次，如果while裡面的條件式結果為true，它會繼續執行do{}裡面的程式，直到 while 條件式為false。語法：do{statement1; statement2; ...}while(expression);
以下範例利用do{}while()來計算1+2+3+4+5。
var i=1; var sum=0; do{sum+=i; i+=1;}while(i<=100); alert(sum);
補充範例如下： 
window.onload = function(){
	var inputArray = new Array();
	var arrayIndex = 0;
	var inputData;
	var inputTotal = 0;
	var returnMsg = "您輸入了：";
	
	do{
		inputData = prompt("請輸入一個數值，或輸入0退出",0);
		inputArray[arrayIndex]=inputData;
		arrayIndex+=1;
	}while(inputData!=0);
	
	for(var i=0; i < inputArray.length;i++){
		returnMsg+=inputArray[i]+"，";
		inputTotal+=Number(inputArray[i]);
	}
	
	returnMsg+="總和："+inputTotal;
	document.write(returnMsg);
}
for-in 關鍵字 for...in通常用來列舉物件的屬性。語法如下： 
for(key in object){statement;}
它用來瀏覽 object 中的所有屬性，並且每一個屬性都執行一次 statement 循環體；例如我們需要列出 window 物件的所有屬性，儘管並不知道 window 物件到底有多少屬性，以及每個屬性相對應的名稱，但透過 for...in 語句便可以很輕地獲得各種參數如下例將列出 window 的所有物件： 
for(var key in window){
	document.write(key+": "+window[key]+"<br />"); // 一次列出所有window的屬性及值
}
範例2：用於自訂的物件，如下例依序傳回每個陣列元素的值： 
var mycars = [ "Saab", "Volvo", "BMW" ];
for(x in mycars){alert(mycars[x]);} //依序傳回每個陣列元素的值

var prices = { apple: 15, pen: 18, pinapple: 120 }
for(var key in prices){
	alert(key+": "+prices[key]); // 分次alert出prices的鍵及值
}
我們也可以將for...in寫成一個自訂函數方便調用，如下例： 
function listMember(targetObj){
	var s = "";
	for(key in targetObj){s += key + ": " + targetObj[key] + "<br>";}
	return s;
}
var main = {a:123, b:456, c:789}
document.write(listMember(main));
/*傳回：
a: 123
b: 456
c: 789*/
for-of 關鍵字 for-of語句是ES6新增，用法和for-in很像
它可使用於Array, Object, Set, Map
for-in取的是鍵，而for-of取的是值： 
var todo = [
	{id: 0, text: "剪頭髮", finish: false},
	{id: 1, text: "打東東", finish: true}
];

for(var i in todo){ //for/in語句取的是「鍵」
	console.log(todo[i].text);
}
// 剪頭髮
// 打東東

for(var v of todo){ //for/of語句取的是「值」
	console.log(v.text);
}
// 剪頭髮
// 打東東
下例示範在Javascript物件中使用for-of： 
var Obj = { firstname: "Chang", lastname: "JOHN" };
for (var key of Object.keys(Obj)) {
	console.log(key + ": " + Obj[key]);
}

break 關鍵字 使用break可以立即跳出迴圈，阻止再次執行循環體中的任何程式碼，或是像switch條件式中斷程式區塊。
範例1： 
var sum = 0;
for(var i=0; i<5; i++){
	if(i > 2){
		alert('迴圈要停了');
		break;
	}
	sum += i;
	alert('i='+i);
}
alert('總合是：'+sum);
範例2： 
window.onload = function(){
	var inputArray = new Array();
	var arrayIndex = 0;
	var inputData;
	var inputTotal = 0;
	var returnMsg = "您輸入了：";
	
	do{
		inputData = prompt("請輸入一個數值，或輸入0退出",0);
		if(isNaN(inputData)){
			document.write("您最後輸入的資料不是數字<br />");
			break;
		}
		inputArray[arrayIndex]=inputData;
		arrayIndex+=1;
	}while(inputData!=0);
	
	for(var i=0; i < inputArray.length;i++){
		returnMsg+=inputArray[i]+"，";
		inputTotal+=Number(inputArray[i]);
	}
	
	returnMsg+="總和："+inputTotal;
	document.write(returnMsg);
}
continue 關鍵字 退出一次的迴圈，根據控制運算式還允許進行下一次迴圈。
範例如下： 
window.onload = function(){
	var inputArray = new Array();
	var arrayIndex = 0;
	var inputData;
	var inputTotal = 0;
	var returnMsg = "您輸入了：";
	
	do{
		inputData = prompt("請輸入一個數值，或輸入0退出",0);
		if(isNaN(inputData)){
			document.write("您最後輸入的資料不是數字<br />");
			continue;
		}
		inputArray[arrayIndex]=inputData;
		arrayIndex+=1;
	}while(inputData!=0);
	
	for(var i=0; i < inputArray.length;i++){
		returnMsg+=inputArray[i]+"，";
		inputTotal+=Number(inputArray[i]);
	}
	
	returnMsg+="總和："+inputTotal;
	document.write(returnMsg);
}
