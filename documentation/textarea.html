Javascript - textarea物件

名稱 類形 說明
textarea 物件 The Textarea object represents a text-area in an HTML form.
For each <textarea> tag in an HTML form, a Textarea object is created.
You can access a Textarea object by indexing the elements array (by number or name) of the form or by using getElementById().
可輸入文字資料的表單元素有text、password、textarea，如果我們希望限制使用者輸入字元的長度，我們可以使用HTML表單元素中的maxlenght屬性，text和password都可以輕鬆做到這一點，但有許多瀏覽器的<textarea>標籤並不支援HTML的maxlength屬性，這時候我們就必須為這些不支援的瀏覽器的maxlength屬性加上自訂的textareaMax()函式；下例(1)為模擬maxlength的行為，如果監聽onkeypress事件時textareaMax()傳回false的時候，使用者將不能再輸入文字(Enter也算一個字元)；
這個範例雖然有參考價值，但使用時還是要再加以修改強化，因為textareaMax()還是有一些bug，例如有的瀏覽器當達到指定字數時，Backspace會不能按(因為onkeypress)，如此就不能修改資料，另外如果是用貼上的方式來輸入資料，有的瀏覽器就算超過字數也照貼不誤，因此正式撰寫時這些問題都要考慮再加以處理。
範例1： 
<form><textarea maxlength="5"></textarea></form>
<script>
	function textareaMax(){return this.value.length<=this.getAttribute('maxlength');}
	var myTextarea = document.getElementsByTagName('textarea')[0];
	myTextarea.onkeypress = textareaMax;
</script>
accessKey 屬性 String value that sets/ returns the accessKey for the field.
cols 屬性
defaultValue 屬性 元性初始設定的value值
disabled 屬性 Boolean value that sets/ returns whether the field is disabled.
form 屬性 指向表單元素所在的<form>
References the form that contains the textarea.
name 屬性 讀取/設定表單元素name屬性的值
Sets or returns the value of the name attribute of a form element.
readOnly 屬性
rows 屬性
type 屬性 設定/讀取表單元素type屬性的值；IE8(含)以下的瀏覽器只能讀取，不支援設定屬性的功能。
A property available on all form elements, "type" returns the type of the calling form element, in this case, "Textarea".
value 屬性 存/取表單元素value屬性的值
Read/write string that specifies the value of the textarea.
onBlur 事件屬性 Code is executed when the focus is removed from the textarea.
onChange 事件屬性 Code is executed when the user changes the value within the textarea, and removes focus away from the field.
onFocus 事件屬性 Code is executed when the focus is set on the textarea.
onKeyDown 事件屬性 Code is executed when user presses down the key within the textarea.
onKeyPress 事件屬性 Code is executed when user presses the key within the textarea.
onKeyUp 事件屬性 Code is executed when user releases a key within the textarea.
onSelect 事件屬性 Code is executed when user selects some text within the textarea.
blur() 方法 使焦點離開某個元素
Removes focus away from the textarea.
focus() 方法 聚焦到某個元素上
Sets focus on the textarea.
範例： 
<form>
	<textarea name="textarea01"></textarea>
</form>
<script>
	window.onload = function(){
		document.forms[0].textarea01.focus();
	}
</script>
select() 方法 選取元素內的文字
Highlights the content of the textarea.
範例： 
<form>
	<textarea name="textarea01">select</textarea>
</form>
<script>
	document.forms[0].textarea01.onmouseup = function(){
		this.select();
	}
</script>
