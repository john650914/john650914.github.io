Javascript - Conditionals

名稱 類形 說明
if 關鍵字 if條件式，以下是簡易語法示範。
if(條件判斷){如果條件成立就執行這裡的程式;}
實測發現，在Javascript if條件式中，小括弧內的判斷式為true，或是無程式錯誤產生都算成立，請參考以下各範例： 
if(1+1==2){alert('true');} //成立

if(document.getElementsByTagName){alert('true');} //成立

if(1+1==3){alert('true');} //不成立

if(document.getElementByTag){alert('true');} alert('continues'); //不成立：列舉不存在的方法，大括弧內的程式不執行，但不列為程式錯誤，只認定為條件式不成立，接來下的程式會繼續執行；如果條件改成"!document.getElementByTag"則條件式成立。

補充：如果我們alert(document.getElementsByTagName);會傳回完整的函數，而不是true；而alert(1+1==2);會傳回true。
if/else 關鍵字 配合if條件式，以下為簡易語法示範。
語法1：if(ans_txt.text == 'apple'){result_txt.text='真棒';} else{result_txt.text='再試一次';}
語法2：if(x==y){平手;} else if(x>y){x勝;} else if(x<y){y勝;}
switch-case-break-default 關鍵字 當條件式需要判斷的情況比較多的時候，通常採用switch語句來實現，語法如下：
switch(expression){
　case value1: statement1
　　break;
　case value1: statement1
　　break;
　　...
　case valuen: statementN
　　break;
　default: statement;
}
如果expression的值等於某個case的值則執行相對應的statement；break關鍵字當條件符合且執行完statemant後用來跳出switch條件式，如果沒有關鍵字break，程式碼就會繼續進入下一個情況；當switch條件式找不到符合的case，就執行default內的程式。
下例示範判斷性別的switch條件式。 
var myGender = prompt("請輸入'男'or'女'");
switch(myGender){
　case "男": alert("您是男生");
　　break;
　case "女": alert("您是女生");
　　break;

　default: alert("輸入錯誤");
}
