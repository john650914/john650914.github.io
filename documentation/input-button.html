Javascript - input(button)物件

名稱 類形 說明
accessKey 屬性 String value that sets/ returns the accessKey for the button.
disabled 屬性 Boolean value that sets/ returns whether the button is disabled.
form 屬性 指向表單元素所在的<form>
References the form that contains the reset button.
name 屬性 讀取/設定表單元素name屬性的值
Sets or returns the value of the name attribute of a form element.
type 屬性 設定/讀取表單元素type屬性的值；IE8(含)以下的瀏覽器只能讀取，不支援設定屬性的功能。
A property available on all form elements, "type" returns the type of the calling form element, in this case, "button".
value 屬性 讀取表單元素value屬性的值
A read only string that specifies the value of the button.
onClick 事件屬性 Code is executed when user clicks on the button.
blur() 方法 使焦點離開某個元素
Removes focus away from the button.
click() 方法 模擬使用者按一下該元素
Simulates a user clicking on the form button.
focus() 方法 聚焦到某個元素上
Sets focus on the button.
