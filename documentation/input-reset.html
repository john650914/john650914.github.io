Javascript - input(reset)物件

名稱 類形 說明
disabled 屬性 Boolean value that sets/ returns whether the reset button is disabled.
form 屬性 指向表單元素所在的<form>
References the form that contains the reset button.
name 屬性 讀取/設定表單元素name屬性的值
Sets or returns the value of the name attribute of a form element.
type 屬性 設定/讀取表單元素type屬性的值；IE8(含)以下的瀏覽器只能讀取，不支援設定屬性的功能。
A property available on all form elements, "type" returns the type of the calling form element, in this case, "reset".
value 屬性 讀取表單元素value屬性的值
A read only string that specifies the value of the reset button.
onClick 事件屬性 Code is executed when user clicks on the reset button.
