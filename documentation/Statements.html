Javascript - Statements

名稱 類形 說明
var 關鍵字 宣告變數時使用，在Javascript裡變數的宣告是很自由的，它可以是任何資料類形，包括沒有內容的變數，就連變數的形態都不用你去操心，Javascript使用的是弱類型的變數，它能夠自行去判別適合的形態，這一點就和C語言完全不同，以下是幾個變數的宣告：
var a;
var b=0;
var c='alex';
var a=0, b='box', c=false; //同時宣告多個變數的寫法
var s=new String(); //預先宣告變數的資料類型為字串，也可寫成：var s="";
以上語法把var刪除也可以執行，但意義上是不同的，沒有使用var關鍵字宣告的變數將視為全域變數，不管是在函式內或函式外。另外，變數的名稱需遵循以下3條規則：
  • 第一個字元必須是字母、底線(_)或是美元符號($)，不可是數字
  • 餘下的字母可以是底線、美元符號、任意字母或是數字字元
  • 變數名不能是關鍵字或者保留字
當字串過長時，JavaScript允許字串的換行宣告，有兩種方式：
1.用"+"號來做換行的宣告 
var showstring="I am Alex! I like JavaScript very much!"
+"And how about you?"
2.用不可見字元"\"來做換行的宣告 
show="I am Alex!I like JavaScript very much!\
And how about you?"
with 關鍵字 用來宣告常用的預設物件，可以縮短使用物件成員的字串。通常用在使用物件頻繁的區塊。使用時要注意變數名稱與成員名稱不要相同。
語法：with(object){statements}
以下範例(1)的"document.form1"就是一個預設物件，而{}內的"field1"指的就是"document.form1.field1"： 
with (document.form1){
	field1.value="test"
	field2.value="test2"
}
範例(2)指定 Math 物件為預設的物件。with 語法裡面使用了 PI 屬性和 cos、sin 方法的語句，而無需指定物件。JavaScript 會假定他們參照的是 Math 物件： 
var a, x, y;
var r = 10;
with (Math) {
	a = PI * r * r;
	x = r * cos(PI);
	y = r * sin(PI/2);
}
附註︰雖然使用 with 語句可以使你的程式更加簡潔，但不當使用 with 也會在一定程度上使你的程式速度變慢。
