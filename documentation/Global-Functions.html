encodeURI() 方法 將傳入的字串轉換成UTF-8編碼的字元。
Javascript中編碼的方法有3個(escape、encodeURI、encodeURIComponent)，一般情況下，最好選擇encodeURIComponent()

encodeURI()的特性
  • 不編碼符號包括：~!@#$&*()=:/,;?+'
  • 用encodeURI()會比使用escape()轉換URI正確。比較要注意的是這個方法不處理'(單引號)。
encodeURIComponent() 方法 將傳入的字串轉換成UTF-8編碼的字元。
Javascript中編碼的方法有3個(escape、encodeURI、encodeURIComponent)，一般情況下，最好選擇encodeURIComponent()

以下是encodeURIComponent()的特性
  • 不編碼符號包括：~!*()'
  • encodeURIComponent()是適合大部分單一 URI的編碼方法。這個方法通常可以正確的編出正確的編碼，例如要編碼一段有flash的html(參數有&符號)，如果用其他的方法，大概解碼後flash也不能看了。比較要注意的是這個方法也不處理'(單引號)。

範例如下： 
url = url + "?name=" + encodeURIComponent("中文");
escape() 方法 將傳入的字串轉換成UTF-8編碼的字元。
Javascript中編碼的方法有3個(escape、encodeURI、encodeURIComponent)，一般情況下，最好選擇encodeURIComponent()

escape()的特性
  • 不編碼的符號包括：@*/+
  • escape 不編碼+(加號)，將使表單欄位資料中有空白的處理(用+連結字串)造成問題。且escape在處理非ASCII語系的字元會有問題。所以，除非特例，應盡量避免使用escape()。

isNaN() 方法 返回一個Boolean值，判斷參數是否不是一個數值，包括整數和浮點數，NaN = Not a Number；如果參數是字串，isNaN()會先檢查字串內容是不是數字，不是數字才返回true。
以下範例將返回false、true、false。
例：alert(isNaN("1080")); alert(isNaN("john")); alert(isNaN(1080));
Number() 方法 將指定的對象轉換為數值，無論是整數還是浮點數。
以下範例(1)將傳回把現在時間轉為數值資料的數值。
例1：var myDate = new Date(); alert(Number(myDate));
範例(2)將示範字串的轉換將傳回1080及10.8及NaN
例2：alert(Number("1080")); alert(Number("10.80")); alert(Number("10 80"));
parseFloat() 方法 將字串轉換為數值類型，parseFloat()將字串轉為浮點數。只有字元類型才能使用此方法，否則直接返回NaN(not a number)。
parseFloat()會先分析指定的字串，首先檢查位置0的字元是否為有效數字，如果不是則返回NaN，不再進行任何操作。如果位置0為有效數字，則繼續往下找到字串結束或是遇見非數值字元為止，如下例： 
alert(parseFloat("34535orange")); //返回34535
alert(parseFloat(435.34)); //返回435.34
alert(parseFloat("435.34")); //返回435.34
alert(parseFloat("435.34.564")); //返回435.34
alert(parseFloat("john650914@gmail.com")); //返回NaN
alert(parseFloat("0xA")); //parseFloat()沒有預設進制，直接輸出第一個字元"0"
parseInt() 方法 將字串轉換為數值類型，parseInt()將字串轉為整數。只有字元是數值才能使用此方法，否則直接返回NaN(not a number)。
parseInt()會先分析指定的字串，首先檢查位置0的字元是否為有效數字，如果不是則返回NaN，不再進行任何操作。如果位置0為有效數字，則繼續往下找到字串結束或是遇見非數值字元為止，如下例： 
alert(parseInt("4567red")); //返回4567
alert(parseInt("4567red89")); //返回4567
alert(parseInt("53.5")); //返回53
alert(parseInt("john650914@gmail.com")); //返回NaN
另外parseInt()方法還在轉換為數值的同時執行進制的轉換，如下例： 
alert(parseInt("0xC"));  //返回12
alert(parseInt("AF",16)); //返回175
alert(parseInt("11",2)); //返回3
alert(parseInt("011")); //返回9
alert(parseInt("011",8)); //返回9
alert(parseInt("011",10)); //返回11

toString() 方法 將array、number、boolean等等的Javascript物件轉換為字串物件。範例如下： 
var number01 = 123;
alert(typeof(number01)); //返回number
alert(typeof(number01.toString())); //返回string

var a=b=c=4;
alert(a+b+c.toString()+8); //返回848

var myArray = ['a','b','c'];
alert(typeof(myArray.toString())); //返回string
另外toString()方法還可加入參數進行進制的轉換，如下例： 
var a=11;
alert(a.toString(2)); //返回1011
alert(a.toString(3)); //返回102
alert(a.toString(8)); //返回13
alert(a.toString(16)); //返回b
補充：直接為某個數值用+號加上一個長度為零的空字串，可以快速將數值轉為字串，如下例： 
var number01 = 123;
alert(typeof(number01+'')); //返回string
