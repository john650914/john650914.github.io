Javascript - Boolean物件

名稱 類形 說明
Boolean 物件 Javascript中的布林物件只有true和false兩種值；從某種意義上來說，電腦程式設計就是跟布林值打交道，電腦就是0和1的世界。
下例用typeof做一個簡單的觀察它的用法： 
var myBoolean = true;
alert(typeof(myBoolean)); //返回boolean物件
var myBoolean = 'true';
alert(typeof(myBoolean)); //返回string物件
