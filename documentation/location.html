Javascript - location物件

執行GET請求的URL中問號(?)後的部份。
名稱 類形 說明
location 物件 location物件是window和document物件的屬性，主要作用是分析和設置頁面的URL位址。
hash 屬性 如果URL包含書籤#，則返回該符號後的內容。
例：#section1
host 屬性 Specifies the hostname and port (if available) of a URL. (ie: "www.mysite.com" or "www.mysite.com:563").
hostname 屬性 Specifies the hostname portion of the URL (ie: "www.mysite.com").
href 屬性 現在載入完整的URL。
例：http://picasaWeb.google.com/isaacshun/LightningOverBeijing
用於獲得或設置視窗的URL，是location最常用的屬性，類似於document的URL屬性。改變此屬性的值就可以直接導航到新的頁面，如下例：
location.href="http://picasaWeb.google.com/isaacshun";
經測試後發現location.href對各個瀏覽器的相容性很好，而且新URL會記錄到瀏覽器的瀏覽歷史中，表示可以執行回上一頁的操作。
另外要注意的是如果location.href="...";還有其它的程式是會繼續執行的。
pathname 屬性 URL中主機名稱後的部份。
例：/isaacshun/LightningOverBeijing
port 屬性 URL中請求的埠號。
例：8021
protocol 屬性 URL使用的協定。
例：http:
search 屬性 返回GET請求的URL中問號(?)後的部份。
例如http://www.yahoo.com.tw?keyword=isaac&randomNumber=19820624將傳回：?keyword=isaac&randomNumber=19820624
reload() location方法 用來重新載入頁面，此方法接受一個布林值，如果是false則從瀏覽器的暫存中重載，如果為true則從伺服器上重載，預設值為false。下例則執行從伺服器上重載。
例：location.reload(true);
replace() location方法 用於設置目前視窗新的URL位置，類似location.href屬性，不同的是此方法不提供回上一頁的操作，像安全級別較高的銀行系統等就會需要這個方法。
下例運用了setTimeout()方法清楚的顯示頁面轉跳的效果，換頁後會發現瀏覽器的"回上一頁"按鈕失效了。
例：setTimeout(function(){location.replace("http://www.google.com");},2000);
