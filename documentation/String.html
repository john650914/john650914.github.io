Javascript - String物件

名稱 類形 說明
String 物件 字串是由零個或多個字元構成，typeof()的傳回值是string。字元可以是字母、數字、標點符號和空格。
字串必須放在單引號或是雙引號裡，如字串裡有單引號則把整個字串用雙引號包起來，反之亦然；
還有當我們輸出的字串中，有含一些指令會用到的特殊字元時，要把該字元當一般字串輸出，就需要在前面用反斜線\讓該字元跳脫特殊字元，變成一般字串值來輸出，上述各種狀況如下例所示： 
var sentence = "let's go";
var case = 'the number "2001011026"';
var score = "run time 3'15\""; //字串裡的"如果不加\會被視為不正常的結尾導致瀏器錯誤

length 屬性 字串具有length屬性，它返回字串中字元的個數。
以下範例將返回10。
例：var myString = "hello! 哈囉!"; alert(myString.length);

既然字串有length屬性，那它也可以用索引值的方式取得某個字元，例如： 
var myStr = 'ABC';
console.log(myStr[1]); //B

indexOf() 方法 搜索字串的方法，返回指定文字於字串的開始位置(index)，字串中的第一個字元的位置為0，如果找不到則返回-1，並且區分大小寫；可接受兩個參數，第一個為欲搜索的文字，第二個為指定從第幾個字元的位置開始搜索，範例如下： 
var myString = "Tsinghua University";
alert(myString.indexOf("ing")); //返回2
alert(myString.indexOf("i",5)); //返回11
alert(myString.indexOf("inc")); //返回-1
alert(myString.indexOf("ING")); //返回-1
帶入空的字串將傳回0： 
var myStr = 'This is a string.';
alert(myStr.indexOf('')); //傳回0
判斷字串中是否包含某些字元： 
var myString = 'http://www.yahoo.com.tw';
if(myString.indexOf('http://')==-1){
	alert('myString不是一個完整的網址');
}
else{
	alert('myString是一個完整的網址');
}
lastIndexOf() 方法 和indexOf()功能相同，但lastIndexOf()是由字串的結尾往前搜。
搜索字串的方法，返回指定文字於字串的開始位置(index)，字串中的第一個字元的位置為0，如果找不到則返回-1，並且區分大小寫；可接受兩個參數，第一個為欲搜索的文字，第二個為指定從第幾個字元的位置開始往回搜索，範例如下： 
var myString = "Tsinghua University";
alert(myString.lastIndexOf("ing")); //返回2
alert(myString.lastIndexOf("i")); //返回16
alert(myString.lastIndexOf("i",3)); //返回2
alert(myString.lastIndexOf("ING")); //返回-1
match() 方法 這是在一句中找某個字母或文字，語法如下：
str.match("目標字")
str 是有關的字串，目標字是要在這字串內尋找的文字，找到就傳回目標字，找不到就傳回 null 這一個字，請看這例子： 
x="My favourite colors include red, yellow and blue.";
alert(x.match("red"));
以上的 script 會在 alert 對話盒顯示 red 這個找到的字，假若將上述的 scritp 改為以下內容： 
x="My favourite colors include white, yellow and blue.";
alert(x.match("red"));
alert 對話盒會顯示 null 這字，表示在 x 內沒有 red 這個字。

replace() 方法 replace()方法用於在字串中用一些字元替換另一些字元，語法如下：
語法：stringObject.replace(substr,replacement)
str 是有關的字串，"目標字" 是要在這字串內尋找的文字，找到就用 "代替字" 來取代，並傳回新的字串，原有字串不變，範例如下： 
x="My favourite colors include red, yellow and blue.";
alert(x.replace("red","violet"));
以上的 script 是在 x 這句內以 violet 取代 red, alert 對話盒顯示這句：My favourite colors include violet, yellow and blue.。若 x 內沒有目標字就不會被改變。
規則運算式(RegExp用法請見"RegExp物件")
search() 方法 這是在一句中找某個字母或文字，使用以下語法：
str.search("目標字")
str 是有關的字串，目標字是要在這字串內尋找的文字，找到就傳回目標字的排序位置，找不到就傳回 -1，請見下例：
x="My favourite colors include red, yellow and blue.";
alert(x.match("red"));
以上的 script 會在 alert 對話盒顯示 28，表示 red 字是在 x 這句中的第 28 個位置 (從 0 開始計算)，若找不到就傳回 -1。
規則運算式(RegExp用法請見"RegExp物件")
slice() 方法 從字串中取出其中的一段文字，可輸入兩個參數，分別為起始位置和終止位置，但不包含終止位置的那個字元，第一個字元的位置為0；如果只設置一個參數則視為取得該參數的位置到字串的結尾，範例如下： 
var myString = "Tsinghua University";
alert(myString.slice(1,3)); //返回si
alert(myString.slice(4)); //返回ghua University
slice()對於負數的處理方式和substring()不同；如果參數為負值，slice()會從字串的結尾往前計算，如下例： 
var myString = "Tsinghua University";
alert(myString.slice(2,-3)); //返回inghua Univers
split() 方法 split()是將字串轉換為陣列的方法；我們知道使用toString()或join()可以將陣列轉為字串，相反地也有將字串轉為陣列的方法，語法如下： 
stringObj.split("分隔標示",[設定字元範圍]);
當指定一個沒有字元的分隔標示則會把每個字母拆成各別的陣列元素，並以逗點作分隔；各種使用範例如下： 
var countries = 'Taiwan USA China';
alert(countries.split("")); //傳回T,a,i,w,a,n, ,U,S,A, ,C,h,i,n,a
alert(countries.split("",5)); //傳回T,a,i,w,a
alert(countries.split(" ")); //傳回Taiwan,USA,China
var myArray = countries.split(" "); //先將split的結果存入一個陣列變數
alert(myArray[0]); //傳回Taiwan
alert(myArray[0]+'-'+myArray[1]+'-'+myArray[2]); //傳回Taiwan-USA-China
substr() 方法 從字串中取出其中的一段文字，可輸入兩個參數，分別為起始位置和字串長度。
以下範例將返回ing。 
var myString = "Tsinghua University"; alert(myString.substr(2,3));
substring() 方法 從字串中取出其中的一段文字，可輸入兩個參數，分別為起始位置和終止位置，但不包含終止位置的那個字元，第一個字元的位置為0；如果只設置一個參數則視為取得該參數的位置到字串的結尾。
另外substring()還有一個特性，兩個參數比較小的會自動往前移做為起始位置，例如substring(2,1);就會自動轉成substring(1,2);，範例如下： 
var myString = "Tsinghua University";
alert(myString.substring(1,3)); //返回si
alert(myString.substring(4)); //返回ghua University
substring()對於負數的處理方式和slice()不同；如果參數為負值，substring()會把負數忽略做為0處理，並將兩個參數中較小的作為起始位置，如下例： 
var myString = "Tsinghua University";
alert(myString.substring(2,-3)); //返回Ts(-3變成0再變成第一個參數，2變成第二個參數)

trim() 方法 Trims a string on both sides for any spaces and returns the result. The original string is unmodified: Example: 
var sitename=" JavaScript Kit   "
alert(sitename.trim()) //alerts "JavaScript Kit". Variable itself unchanged.
Since trim() is not yet supported in all browsers (most notably IE), it should be coupled with a fall back method for achieving the same result in those lesser browsers, such as via regular expressions. The following example uses trim() in capable browsers, and regular expressions instead in non capable to trim a string:

Cross Browser Example: 
function trimstring(str){
	if (str.trim)
		return str.trim()
	else
		return str.replace(/(^\s*)|(\s*$)/g, "") //find and remove spaces from left and right hand side of string
}
