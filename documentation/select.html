Javascript - select物件

名稱 類形 說明
select 物件 Javascript中的select物件代表HTML中的<select>標籤，以下拉選單的方式呈現，它是表單元素中比較特殊的一個類別，透過<option>來設置所有列表項。當它是單選時功能相當於<radio>元素，而當它是多選時(<select multiple="multiple">)功能相當於<checkbox>元素，但是所佔用的頁面面積卻小很多。
option物件詳細說明請見option目錄。
disabled 屬性 Boolean value that sets/ returns whether this SELECT element is disabled.
form 屬性 指向表單元素所在的<form>
References the form that contains the selection list in question.
length 屬性 表示<option>選項的個數。
Returns the number of options in the SELECT element: 
document.getElementById("mymenu").length //returns number of options
multiple 屬性 Boolean that indicates whether this SELECT allows more than one option to be selected simultaneously.
name 屬性 讀取/設定表單元素name屬性的值
Sets or returns the value of the name attribute of a form element.
options[] 屬性 擷取選項的陣列，例如select01.options[2]表示該下拉式功能表中的第3個option項目。 An HTMLCollection of Options objects containing each option within this SELECT element.
selectedIndex 屬性 被選中的選項序號，如果沒有選項被選中則為-1。對於多選下拉式功能表而言，返回被選中的第一個選項序號。從0開始計數。
下拉選單中最重要的就是取得被使用者所選中的項目，我們可以透過selectedIndex屬性輕鬆地訪問到被中的項目，如下範例(1)： 
<form>
	選擇您的血型：
	<select>
		<option value="type O">O型</option>
		<option value="type A">A型</option>
		<option value="type B">B型</option>
		<option value="type AB">AB型</option>
	</select>
	<button>顯示選中的項目</button>
</form>
<script>
	var select01 = document.forms[0].elements[0];
	var button01 = document.forms[0].elements[1];
	button01.onclick = function(){
		var selIndex = select01.options.selectedIndex;
		alert(select01.options[selIndex].text);
		return false;
	}
</script>
以上範例只針對單選的下拉選單，覆選的做法請參考option物件selected的範例。
An integer that gets or sets the index of the selected option. If non is selected, -1 is returned. If multiple options are selected, it returns the index of the first selected option. When setting this property, the OPTION at the given position will be selected (while any previously selected option deselected).
The following example alerts the index number of the selected option within a SELECT element when the user makes a selection:
Example(s): 
<script type="text/javascript">
function alertselected(selectobj){
 alert(selectobj.selectedIndex)
}
</script>

<form id="someform">
<select id="sample" onChange="alertselected(this)">
<option>Option 1</option>
<option>Option 2</option>
<option>Option 3</option>
</select> 
<input type="button" onClick="alertselected(document.getElementById('sample'))" value="alert" />
</form>
size 屬性 An integer reflecting the number of OPTIONs being displayed at once within this SELECT element.
type 屬性 設定/讀取表單元素type屬性的值；IE8(含)以下的瀏覽器只能讀取，不支援設定屬性的功能。
在select物件裡返回的是下拉功能表的類型。單選返回字串"select-one"，多選返回字串"select-multiple"。

範例1，傳回select物件單選或是複選的值： 
<form>
	<select>
		<option value="ball">option 01</option>
		<option value="music">option 02</option>
		<option value="climb">option 03</option>
	</select>
	<select multiple="multiple" size="3">
		<option value="ball">option 01</option>
		<option value="music">option 02</option>
		<option value="climb">option 03</option>
	</select>
</form>
<script>
	var select01 = document.forms[0].elements[0];
	var select02 = document.forms[0].elements[1];
	alert('選單1的類型是：'+select01.type+'\n選單2的類型是：'+select02.type);
</script>
範例2，為表單中所有的<select>元素加上"selectclass"的class name： 
var myform=document.getElementById("someform");
for(var i=0; i<myform.elements.length; i++){
	if(myform.elements[i].type=="select-one" || myform.elements[i].type=="select-multiple"){
		myform.elements[i].className="selectclass";
	}
}
onBlur 事件屬性 Code is fired whenever the focus is removed from the SELECT element.
onChange 事件屬性 Code is fired whenever an OPTION within SELECT changes, through the user selecting another option. This event handler is commonly used to create a SELECT element that reacts as soon as the user has selected an option manually. Here's a SELECT menu that navigates to a particular site upon user selection:
Example(s): 
<form id="aform">
<select id="mymenu" size="1">
<option value="nothing" selected="selected">Select a site</option>
<option value="http://www.dynamicdrive.com">Dynamic Drive</option>
<option value="http://www.codingforums.com">Coding Forums</option>
<option value="http://www.cssdrive.com">CSS Drive</option>
</select>
</form>

<script type="text/javascript">

var selectmenu=document.getElementById("mymenu")
selectmenu.onchange=function(){ //run some code when "onchange" event fires
 var chosenoption=this.options[this.selectedIndex] //this refers to "selectmenu"
 if (chosenoption.value!="nothing"){
  window.open(chosenoption.value, "", "") //open target site (based on option's value attr) in new window
 }
}
</script>
onFocus 事件屬性 Code is fired whenever the focus is set on the SELECT element.
add()(newoption, beforeoption*) 方法 Dynamically adds a new OPTION to the SELECT element, where:
  • newoption: The new option to add, created using new Option().
  • beforeoption*: This parameter differs depending on the browser used:
    • In non IE browsers, an existing option (reference) within SELECT in which to add the new option following it. If null is entered, the new option is added at the very end.
    • In IE browsers (including IE8 beta2), an integer representing the position of the existing option to add the new option to (ie: 0=add following 1st element, 1=add following 2nd element etc). Leave this parameter out entirely to add the new option to the very end.
Due to the differences in the implementation of the 2nd parameter of add(), to make it cross browser, you can use try/catch to see which version the browser accepts. The 1st parameter newoption should simply be a new option created using new Option() (see Option object below for more info).
Example(s): 
<form id="someform">
<select id="sample">
<option value="1">Option 1</option>
<option value="2">Option 2</option>
<option value="3">Option 3</option>
</select>
</form>

<script type="text/javascript">
var myselect=document.getElementById("sample")
try{
 myselect.add(new Option("New Last Option", "4"), null) //add new option to end of "sample"
 myselect.add(new Option("New First Option", "0"), myselect.options[0]) //add new option to beginning of "sample"
}
catch(e){ //in IE, try the below version instead of add()
 myselect.add(new Option("New Last Option", "4")) //add new option to end of "sample"
 myselect.add(new Option("New First Option", "0"), 0) //add new option to beginning of "sample"
}
</script>
results in the SELECT element becoming: 
<select id="sample">
<option value="0">New First Option</option>
<option value="1">Option 1</option>
<option value="2">Option 2</option>
<option value="3">Option 3</option>
<option value="4">New Last Option</option>
</select>
Note1: beforeoption should be an object reference to an option except in IE, where it must be an integer representing the index of the option instead. Note2: An alternate, arguably simpler way of adding a new option to the SELECT element is simply to assign new Option() to one of the options within select.options[]. See Option object below for more info.
remove(optionindex) 方法 Removes an existing option within SELECT at the specified position, where 0=1st option, 1=2nd option etc, for example: Example(s): 
var myselect=document.getElementById("sample")
myselect.remove(myselect.length-1) //removes last option within SELECT
blur() 方法 使焦點離開某個元素
Removes focus away from the selection list.
focus() 方法 聚焦到某個元素上
Sets focus on the selection list.
